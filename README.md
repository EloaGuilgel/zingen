## Sobre o projeto

O Zingen é uma página de apresentação para um app de karaokê, com seções que destacam o produto, funcionalidades, planos e chamada para download. O projeto foi construído com HTML e CSS, priorizando clareza visual, boa organização dos arquivos e uma experiência fluida tanto no desktop quanto no mobile.

## Destaques

- Layout moderno e atrativo para apresentacao do produto.
- Projeto bem estruturado, com estilos separados por responsabilidade.
- Pagina responsiva, adaptada para diferentes dispositivos.
- Organização simples e fácil de manter.
- Publicação pronta para acesso online via GitHub Pages.

## Estrutura

O projeto está dividido de forma clara para facilitar manutenção e evolução:

```text
zingen/
|-- assets/
|-- styles/
|-- index.html
`-- README.md
```

### Organizacao dos arquivos

- `index.html`: estrutura principal da landing page.
- `assets/`: imagens, ícones e elementos visuais do projeto.
- `styles/`: arquivos CSS separados por seções e componentes, deixando o código mais limpo e escalável.

## Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts

## Responsividade

Este projeto foi pensado para oferecer uma boa experiência em diferentes resoluções de tela. A interface se adapta bem ao desktop e ao mobile, mantendo a leitura, o espaçamento e a hierarquia visual consistentes.
